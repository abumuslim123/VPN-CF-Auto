# Подключение мобильных устройств (VLESS)

## Принцип работы

Трафик идёт через Cloudflare CDN — блокировщики видят обращение
к Cloudflare, а не к VPN-серверу. Протокол VLESS + WebSocket + TLS.

## Android

### NekoBox (рекомендуется)
1. Установить [NekoBox](https://github.com/MatsuriDayo/NekoBoxForAndroid/releases) из GitHub
2. Главный экран → "+" → "Импорт из буфера"
3. Вставить `vless://...` ссылку (получить у администратора)
4. **Или**: "+" → "Сканировать QR-код" → навести на QR
5. Нажать кнопку подключения ▶

### Hiddify
1. Установить [Hiddify](https://play.google.com/store/apps/details?id=app.hiddify.com) из Google Play
2. Настройки → Добавить профиль → Добавить из буфера
3. Вставить `vless://...` ссылку
4. **Или**: Сканировать QR-код
5. Подключиться

### V2rayNG
1. Установить [V2rayNG](https://play.google.com/store/apps/details?id=com.v2ray.ang) из Google Play
2. "+" → "Импорт конфига из буфера"
3. Вставить `vless://...` ссылку
4. **Или**: "+" → "Сканировать QR-код"
5. Нажать кнопку подключения внизу экрана

## Выбор ноды

- **ExitNode1** — основная нода (Европа). Используйте по умолчанию.
- **ExitNode2** — резервная нода. Используйте, если основная недоступна.

Обе ноды добавлены в ваш конфиг. Переключение между ними — вручную в приложении.

## Если не работает

1. **Проверьте интернет** — откройте любой сайт без VPN
2. **Попробуйте другую ноду** — переключитесь на ExitNode2
3. **Проверьте время** — системные часы должны быть точными (±2 минуты)
4. **Переустановите конфиг** — удалите профиль и добавьте заново
5. **Обратитесь к администратору** — возможно, домен заблокирован и нужна ротация

## Формат VLESS URI

```
vless://UUID@хост:443?encryption=none&security=tls&type=ws&host=хост&path=%2Fvless-ws&sni=хост&fp=chrome&alpn=h2%2Chttp%2F1.1#Имя
```

- `UUID` — ваш уникальный идентификатор
- `хост` — домен через Cloudflare (например, vpn1-m.example.com)
- `443` — стандартный HTTPS порт
- `type=ws` — WebSocket транспорт
- `security=tls` — шифрование TLS
- `#Имя` — название профиля (видно в приложении)
